# Code of Conduct

This repository aims to be a practical and respectful place for collaborative maintenance.

## Expected Behavior

- Be respectful in issues, pull requests, and reviews.
- Keep technical criticism focused on the work, not the person.
- Cite sources when proposing data corrections or metadata changes.
- Prefer clear, reviewable contributions over large unstructured dumps.

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language
- Fabricated evidence, forged metadata, or intentionally misleading changes
- Sharing private or non-public research material in the repository

## Reporting

If you encounter behavior that makes collaboration unsafe or unproductive, open a private report to the maintainer through GitHub security reporting or a direct repository contact.
