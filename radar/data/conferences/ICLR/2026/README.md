# ICLR 2026 Time-Series Pack

This directory stores the public-source layer for time-series related papers tracked in ICLR 2026.

## Included

- `papers_manifest.json` with paper titles, categories, and public links
- `INDEX.md` generated from the manifest
- `openreview_threads/` with public OpenReview thread captures and structured summaries

## Maintenance Rules

- Keep links traceable to OpenReview or other public sources
- Do not add private notes or unpublished interpretations
- Regenerate derived outputs when the manifest changes
