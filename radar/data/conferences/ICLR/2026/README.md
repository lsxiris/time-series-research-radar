# ICLR 2026 / Time Series / Public Source Pack

这个目录只放**公开资料层**，不放个人研究笔记。

## 当前包含
- 14 篇时间序列相关论文的 OpenReview 论坛链接
- 14 篇论文的 PDF 下载链接
- 14 组对应的审稿 / rebuttal 原始整理（raw / structured）

## 结构
- `papers_manifest.json`：论文清单、论坛链接、PDF 链接、分类
- `openreview_threads/`：审稿线程整理文件

## 使用原则
- 这里只维护公开资料与结构化整理结果
- 不并入私有研究判断、博士级主笔记或内部路线判断
- 后续 topic clustering、主题识别、关键词抽取、method map、benchmark map 都基于这个公开资料层进行
