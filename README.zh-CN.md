# Time Series Research Radar

这是一个面向时间序列研究者的开源研究情报整理项目。

## 这个项目做什么
本仓库希望把时间序列论文追踪做成一条可持续维护的公共工作流，当前包括：
- 会议论文跟踪
- 公开资料层整理
- topic clustering
- method map
- benchmark map
- reviewer / rebuttal 模式整理
- weekly digest 生成

当前主要覆盖：**ICLR / ICML / NeurIPS / KDD**。

## 仓库边界
本仓库只维护**公开资料层**，包括：
- 论文元数据
- 会议 / 年份目录
- PDF / OpenReview / DOI / proceedings 链接
- 公开可得的审稿线程整理
- 基于公开资料形成的索引、标签和地图

本仓库不包含：
- 私人研究笔记
- 未公开实验
- 博士论文内部路线判断
- 非公开注释

## 当前状态
当前已经达到一个可公开发布的 MVP 状态。

已完成：
1. 会议 / 年份目录结构
2. ICLR 2026 公开资料包
3. ICML 2025 时间序列相关论文清单（含标题、链接、摘要、标签）
4. KDD 2025 时间序列相关论文清单（含标题、DOI、track、标签）
5. 多会议索引生成
6. 本地 PDF 批量下载脚本
